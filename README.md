# Musikalisches Fenster Website

Website for a local non-profit built with Astro and Svelte.

To setup locally:

    nvm use
    npm i
    npm run dev

Once a commit is pushed onto the `main` branch, GitHub Actions builds and deploys the site to GitHub Pages.
